Plataformas

Android

Permite crear aplicaciones de Cordova para dispositivos Android y, opcionalmente, usar herramientas de línea de comandos específicas de Android dentro de su flujo de trabajo de desarrollo.

iOS

Tiene un entorno de desarrollo SDK para implementar aplicaciones de Cordova para dispositivos iOS como iPhone y iPad, y cómo utilizar opcionalmente herramientas de línea de comandos centradas en iOS en su flujo de trabajo de desarrollo.

Windows (Obsoleto)

Cordova Windows en Windows 8.1 y Windows Phone 8.1 se basan en Internet Explorer 11 como su motor de renderizado, por lo que, en la práctica, puede usar el potente depurador de IE para probar cualquier contenido web que no invoque las API de Cordova.

OS X (Obsoleto)

Permite configurar su entorno de desarrollo SDK para implementar aplicaciones Cordova para computadoras OS X aunque la platafirma OS X ahora está obsoleta y puede eliminarse de una versión futura de Cordova.

Electron

Electron es un marco que utiliza tecnologías web (HTML, CSS y JS) para crear aplicaciones de escritorio multiplataforma.

¿Qué es Apache Cordova ?

Es un Framework de código abierto que permite crear aplicaciones móviles utilizando HTML, CSS y JavaScript. Ayuda con la gestión de aplicaciones multiplataforma, así como con la integración de complementos de Córdova.

Características

  • Código reutilizable en todas las plataformas
  • Soporte para escenarios offline
  • Acceso a las APIs de dispositivos nativos
  • La línea de comandos se ejecuta en Node.js

Utiliza Apache Cordova cuando se quiera:

  • Tener una aplicación en más de una plataforma, sin tener que re-implementarla con cada lenguaje y herramientas de cada lenguaje.
  • Implementar una app web empaquetada para su distribución en varios portales de tiendas de aplicaciones.
  • Mezclar componentes de aplicaciones nativas con un WebView (ventana especial de un navegador) que pueda accesar a APIs a nivel de dispositivo, o desarrollar una interfaz plugin entre los componentes nativos y el WebView.

Conclusiones

Para utilizar el framework de Cordova fue necesario realizar la descarga de algunos otros complementos que nos ayudaron con la construcción del proyecto y la instalación del mismo framework, primero necesitamos instalar una versión reciente de node.js para poder invocar los comandos node y npm. Para realizar la instalación de Cordova se utilizó el comando npm install -g cordova para windows, no hubo mayores complicaciones con este paso.

Posteriormente se creó el proyecto por medio del comando cordova create actividadcuatrocinco com.actividadcuatro.equipo5 DAHActividadCuatroEquipoCinco, al crear el documento nos marcó que el framework ya se encontraba deprecado para windows, pero nos dejó crear el proyecto de todas formas.

No hubo ningún problema con la instalación ni con la creación del proyecto, Cordova es un framework bastante amigable y poco complejo que puede entenderse y manipularse sin gran conocimiecimiento del framework.
